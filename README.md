##description

Kalkulator sederhana dapat di gunakan untuk melakukan operasi perhitungan dasar

## Requirement

* PHP >=  7
* Bootstrap 4

## Installation 

```bash
masuk folder download
ekstrak file yang di download 
pindahkan ke c:\xampp\htdocs
```

## Usage

start browser

insert browser :
